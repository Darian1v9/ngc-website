# NEW GOLF CLUB Neu-Ulm – Website

Moderne, statische Website für den NEW GOLF CLUB Neu-Ulm (newgolfclub.de).

## Features

- Responsives Design in den Markenfarben (Blau/Orange), Light- & Dark-Mode (🌙/☀️-Button)
- Zweisprachig Deutsch/Englisch (EN/DE-Button, Einstellung wird gespeichert)
- Live-Wetter am Platz über Open-Meteo (kostenlos, kein API-Key)
- Filterbarer Turnierkalender 2026
- Scroll-Animationen und Parallax-Effekte
- SEO: Meta-Tags, schema.org (GolfCourse), sitemap.xml, robots.txt
- Ohne Cookies & Tracking – kein Cookie-Banner nötig

## Struktur

| Datei | Inhalt |
|---|---|
| `index.html` | Startseite mit Platzstatus, Wetter & Platzübersicht |
| `club.html` | Club, Platz, Rating, Greenfee, Mitgliedschaft |
| `kurse.html` | Schnupperkurse, Platzreife, Trainer |
| `turniere.html` | Turnierkalender 2026 mit Filter |
| `lounge.html` | Indoor-Golflounge |
| `cafe.html` | New Golf Café |
| `kontakt.html` | Kontakt & Anfahrt (Google Maps) |
| `impressum.html` / `datenschutz.html` | Rechtliches |
| `css/style.css` | Zentrales Stylesheet (Light/Dark via CSS-Variablen) |
| `js/main.js` | Theme, Sprache, Navigation, Animationen, Wetter |

## Veröffentlichen

Statische Seite – einfach alle Dateien per FTP/Hosting-Panel hochladen.
Optional: Foto als `images/hero-flag.jpg` ablegen → erscheint automatisch als Hero-Hintergrund.
